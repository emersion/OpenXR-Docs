*New Structures*

The following structures are provided to supply supporting runtimes the
necessary information required to work with the EGL API executing on certain
operating systems.

[open,refpage='XrGraphicsBindingEGLKHR',type='structs',desc='The graphics binding structure to be passed at session creation to use EGL',xrefs='xrCreateSession']
--
The slink:XrGraphicsBindingEGLKHR structure is defined as:

include::../../../../generated/api/structs/XrGraphicsBindingEGLKHR.txt[]

.Member Descriptions
****
* pname:type is the elink:XrStructureType of this structure.
* pname:next is code:NULL or a pointer to an extension-specific structure.
* pname:display is a valid sname:EGLDisplay.
* pname:config is a valid sname:EGLConfig.
* pname:context is a valid sname:EGLContext.
****

When creating an EGL-backed slink:XrSession, the application will provide a
pointer to an slink:XrGraphicsBindingEGLKHR structure in the pname:next chain
of the slink:XrSessionCreateInfo.
